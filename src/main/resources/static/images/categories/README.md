# Изображения категорий

Эта папка содержит изображения для обложек категорий товаров.

## Формат файлов
- Формат: PNG или JPG
- Рекомендуемый размер: 400x250px (соотношение 16:10)
- Имена файлов должны соответствовать ID категорий или быть универсальными

## Структура именования
Вы можете использовать два подхода:

### 1. По ID категории (рекомендуется)
- `category-1.jpg` - для категории с ID 1
- `category-2.jpg` - для категории с ID 2
- и т.д.

### 2. По названию категории (альтернатива)
- `rings.jpg` - Кольца
- `earrings.jpg` - Серьги
- `necklaces.jpg` - Ожерелья
- и т.д.

### 3. Placeholder
- `placeholder.jpg` - изображение по умолчанию, если для категории нет своего изображения
